This is a Polymarket trading bot created for personal use. Currently still in progress.
Mainly using CLOB client V2 to trade and using Polymarket Deposit Wallet (signature type = 3) created from EVM (an EOA) as funder address.

Core feature:
1. Create an EOA and create a deposit wallet using it.
2. Deploy the deposit wallet and give it allowance.
3. Create a bridge address to deposit pUSD into deposit wallet.
4. Copy trade single/multiple wallet addresses and specific category of trades to follow for each address.
5. Redeem resolved market winnings.

THIS IS JUST FOR PERSONAL USE AND EDUCATIONAL PURPOSES. PERFORMANCE IS NOT GURANTEED.
